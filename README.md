# Algoritmo_de_Dijkstra

Script que cria uma versão visual de um grafo direcionado usando Busca em Profundidade (DFS) com nós, arestas e uma legenda de cores indicando o estado de cada nó. Inclui uma legenda dinâmica e uma caixa de texto lateral exibindo o histórico de processamento dos vértices.
Código que aplica o algoritmo de Dijkstra em um grafo especifico no código.
