# Comunicação e Redes
Esse git tem meus códigos para a matéria de comunicação e redes(CR) da UFABC.

O primeiro é um Script em python que cria uma versão visual de um grafo direcionado usando Busca em Profundidade (DFS) com nós, arestas e uma legenda de cores indicando o estado de cada nó. Inclui uma legenda dinâmica e uma caixa de texto lateral exibindo o histórico de processamento dos vértices.

O segundo também é um código em python que aplica o algoritmo de Dijkstra em um grafo especificado na arquitetura.
